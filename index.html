Webhooks

Webhooks or HTTP callbacks can be used to get near real time events. It is a lightweight API that powers one-way data sharing triggered by events. Subscribe to the events which you want to listen.

$5

Per 100 events

Benefits

  • Real-Time Data: Webhooks don’t use a request-based system. This option allows your team to view data on a near real-time scale.

  • Supports Automation Efforts: Because data requests are event-based, you don’t have to set up poll timings to your data center. This can help streamline data flow and automation.
